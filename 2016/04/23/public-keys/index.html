mtwabp.fr - Public SSH CA Certificate

The mtwabp.fr network uses SSH certificates to prove the identity of the hosts you can SSH into. For this to work, you’ll have to add the public key of that certificate authority to your ~/.ssh/known_hosts file. The new entry would need to look like this:
